Proyecto de sitio e-commerce creado para el curso de React.js de CoderHouse, a partir de un sitio web en este caso Villa Frida (Tacos Mexicanos Fusión).

En este proyecto para el curso de ReactJS creamos varios componentes con el objetivo de construir un sitio e-commerce funcional. Los componentes centrales para el desarrollo de la funcionalidad son: 
ItemListContainer 
ItemList 
Item 
ItemDetailContainer 
ItemDetail
CartView 
CartWidget 
CartContext 
NavBar
ItemCount.

Comenzamos instalando el software necesario, NodeJS, npm run dev (Ya que el mismo se encuentra realizado con React + Vite), etc.

Para crear la funcionalidad necesaria para un proyecto ecommerce con carrito tuvimos que crear varios componentes tales como:
NavBar 
Footer
CartContext 
CartWidget
Item 
ItemList 
ItemListContainer 
ItemDetailContainer 
ItemDetail 
ItemCount
LoadingSpinner 
CartView 
CheckOut.
